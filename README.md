* 遊戲說明
  * 在一張地圖上會有許多小方塊需要點開，點開小方塊後會顯示一個數字，這個數字就代表著以它為中心的九宮格內藏著幾顆地雷，只要點到地雷就輸了。
  * 踩地雷的獲勝條件是點開所有安全方塊，遊戲中也有「標記」功能可以標記地雷來避免自己誤觸。
